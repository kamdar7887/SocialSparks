# Social Sparks App

## 📱 Description
This Android app provides social interaction suggestions based on the time of day. 
Users can click buttons (Morning, Afternoon, Dinner) to receive a suggestion.

## 🚀 Features
- Morning, Afternoon, and Dinner suggestions
- Reset button
- Simple and user-friendly interface

## 🛠️ Technologies Used
- Kotlin
- Android Studio

## 📋 How to Use
1. Open the app
2. Click any button
3. View suggestion on screen
